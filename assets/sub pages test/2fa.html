Two-Factor Authentication

Autenticação de dois fatores

O 2FA é um procedimento de segurança que garante que serão necessários 2 fatores únicos para liberação de uma ação. O primeiro fator é a senha que o usuário e o segundo pode ser autenticado via token, via detecção de impressão digital, reconhecimento facial, código enviado via sms, entre outros.

O 2FA permite que você:

Envie uma senha via SMS, voz ou e-mail para autenticação do usuário;
Adicione uma camada extra de segurança além da senha pessoal;
Ofereça maior segurança para usuário.

Fortaleça a estratégia de segurança de seu negócio.

Adicionar um número de telefone de recuperação a uma conta individual pode bloquear até 100% do bots automatizados. 99% dos ataques de phishing em massa, e 66% dos ataques direcionados.

Beneficios

1. Ofereça segurança aprimorada para seus clientes.
2. API simples e de rápida implementação.
3. Envio de OTP por meio de vários canais, incluindo SMS, voz ou e-mail.
4. Flexibilidade de canais garante que o usuário conseguirá completar a tarefa desejada mesmo quando tiver problema com um deles. Exemplo: Enviar OTP por SMS, em caso de falha, enviar OTP por chamada de voz, em caso de outra falha, enviar por e-mail.
5. Reduza casos de fraude e invasões e evite o acesso a dados por invasores.
6. Plataforma intuitiva que permite visualizar relatórios de uso por dia, mês ou ano e pesquisar usando diversos critérios de filtro.

Quem usa?

Governo

Acesso ao portal/app. Gestão de informações. Agendamentos. Recuperação de Senha.

varejo

Acesso ao portal/app. Salvar dados de pagamentos. Acesso ao histórico. Recuperação de Senha. Recibo Eletrônico.

Turismo

Acesso ao portal/app. Gestão de Reservas. Acesso ao histórico. Salvar dados de pagamentos. Recuperação de Senha.

Saúde

Acesso ao portal/app. Agendamentos. Tokens de autorização. Telemedicina.

Finanças

Autenticação para transações bancárias. Pagamentos Online (PicPay, PayPal). Digital Wallets (Google Pay, Apple Pay, Samsung Pay).

Usuário acessa seu site ou aplicativo e digita a senha cadastrada para entrar em seu perfil ou completar uma transação.

A telecall recebe a tentativa de login e solicita que o usuário insira seu número de telefone para autorizar o acesso.

Após inserir seu número, a telecall envia para o usuário por SMS, chamada ou e-mail um código PIN do seu único. (Esse código PIN é também conhecido como OTP: One Time Password)

O usuário insere o código no site ou aplicativo para concluir o processo de verificação.
